Microsoft Defender for Office 365 Baseline Configuration

Enterprise-grade security configuration templates for mailbox and collaboration protection

What is This?

This microsite provides production-ready baseline configurations for Microsoft Defender for Office 365 (MDO) in both Standard and Strict protection levels. These configurations follow Microsoft security best practices and can be deployed using Infrastructure as Code (IaC) automation.

Protection Levels

Standard Protection

Recommended for most organizations

  • Anti-phishing with spoofing detection
  • Anti-malware with file type filtering
  • Anti-spam with bulk threshold tuning
  • Safe Attachments scanning
  • Safe Links detonation

Strict Protection

For high-risk environments

  • All Standard features plus:
  • User & domain impersonation protection
  • Aggressive phishing thresholds
  • Quarantine by default actions
  • Automated Investigation & Response (AIR)

Installation Steps

                    
    
                        
  1. 
                            Install Required Modules
                            
    Install-Module ExchangeOnlineManagement -Force -AllowClobber
Install-Module Microsoft.Graph.Authentication -Force
Install-Module Microsoft.Graph.Users -Force
    
                        
    2. 
                        
  2. 
                            Generate Configuration
                            
    Use the Configurator above to generate your organization's configuration file.
    
                        
    3. 
                        
  3. 
                            Validate Configuration
                            
    .\Validate-MDOConfiguration.ps1 -ConfigPath "your-config.json"
    
                        
    4. 
                        
  4. 
                            Test Deployment (Dry Run)
                            
    .\Deploy-MDOBaseline.ps1 `
  -ConfigPath "your-config.json" `
  -TenantId "tenant-guid" `
  -TenantName "Org Name" `
  -TenantDomain "domain.com" `
  -SecurityAdminEmail "admin@domain.com" `
  -DryRun
    
                        
    5. 
                        
  5. 
                            Deploy Configuration

  -TenantName "Org Name" `

Frequently Asked Questions

            
Common questions about MDO baseline configurations


            

                

                    What's the difference between Standard and Strict protection?
                    

                        
Standard Protection is recommended for most organizations. It provides comprehensive protection against common threats including spoofing, malware, and phishing with reasonable thresholds.

                        
Strict Protection is recommended for high-risk organizations such as financial institutions, government agencies, and organizations handling sensitive data. It includes aggressive settings and additional protections like impersonation protection and Automated Investigation & Response.

                    

                


                

                    Can I use Audit mode before enforcing policies?
                    

                        
Yes! Audit mode is recommended for initial deployment. In Audit mode, policies detect threats but don't block or quarantine messages. This allows you to:

                        
    
                            
  • Monitor policy impact
    • 
                            
  • Identify false positives
    • 
                            
  • Adjust settings before enforcement
    • 
                            
  • Review detection reports
    • 
                        

                        
After validation, switch to Enforce mode for active protection.

                    

                


                

                    How do I handle false positives?
                    

                        
If legitimate emails are blocked:

                        
    
                            
  • Review logs in Microsoft Defender portal
    • 
                            
  • Add sender to Allow List via Tenant Allow/Block List
    • 
                            
  • Adjust policy thresholds (e.g., phishing threshold level)
    • 
                            
  • Create exception policies for specific users/groups
    • 
                            
  • Report false positives via Submissions
    • 
                        

                    

                


                

                    What's the cost of implementing these configurations?
                    

                        
These configurations use existing Microsoft Defender for Office 365 features included in your Microsoft 365 subscription. No additional costs for:

                        
    
                            
  • Policy configuration and deployment
    • 
                            
  • Threat detection and enforcement
    • 
                            
  • Safe Attachments/Links scanning
    • 
                            
  • Anti-phishing protections
    • 
                        

                        
Note: Automated Investigation & Response and some advanced features require MDO Plan 2 licenses.

                    

                


                

                    Can I customize these baseline configurations?
                    

                        
Yes! All configurations are fully customizable:

                        
    
                            
  • Modify JSON templates with your preferred settings
    • 
                            
  • Create custom policies after baseline deployment
    • 
                            
  • Adjust thresholds and actions per organizational needs
    • 
                            
  • Add recipient exceptions and allow/block lists
    • 
                        

                        
The baseline serves as a best-practice starting point that can be tailored to your organization.

                    

                


                

                    How do I monitor policy effectiveness?
                    

                        
Use these Microsoft Defender portal tools:

                        
    
                            
  • Configuration Analyzer: Review policy settings vs. recommended baselines
    • 
                            
  • Mail flow reports: Monitor message actions (delivered, quarantined, etc.)
    • 
                            
  • Threat reports: Analyze detected malware, phishing, and spam
    • 
                            
  • Detection tuning: Track and tune automated detections
    • 
                            
  • Submissions: Review user-reported messages
    •
